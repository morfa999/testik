# Схема Header

## Для ЗАЛОГИНЕННЫХ пользователей:
```
┌────────────────────────────────────────────────────────────────────────────────────────────┐
│  🎵 KITSTUDIO   [Тех.поддержка] [Подписка] [Загрузить] [Админ меню*]   🔔 [Avatar]        │
│                        ↑             ↑          ↑            ↑            ↑      ↑          │
│                    (всегда)     (всегда)  (залогинен)  (только админ) (уведом.) (профиль)  │
└────────────────────────────────────────────────────────────────────────────────────────────┘
```

## Для НЕ залогиненных пользователей:
```
┌────────────────────────────────────────────────────────────────────────────────────────────┐
│  🎵 KITSTUDIO   [Тех.поддержка] [Подписка]                          [Войти]  [Начать]      │
│                        ↑             ↑                                 ↑         ↑          │
│                    (всегда)     (всегда)                          (открывает) (регистр.)   │
└────────────────────────────────────────────────────────────────────────────────────────────┘
```

## Элементы Header (слева направо):

1. **Лого** (🎵 KITSTUDIO) - клик возвращает на главную

2. **Навигация (центр)**:
   
   **Всегда видны (даже без авторизации):**
   - 🆘 **Тех.поддержка** - открывает модал отправки репорта
   - 👑 **Подписка** - открывает модал выбора Premium плана
   
   **Только для залогиненных:**
   - ➕ **Загрузить** - открывает модал добавления звука
   - 🛡️ **Админ меню** - видна ТОЛЬКО для `energoferon41@gmail.com`
   
   _При наведении - подчеркивание снизу (без фона)_

3. **Правая часть**:
   
   **Если залогинен:**
   - 🔔 **Уведомления** - выпадающий список (с красной точкой если есть непрочитанные)
   - **Аватар** - открывает профиль пользователя
   
   **Если НЕ залогинен:**
   - **Войти** - открывает модал авторизации
   - **Начать** - открывает модал регистрации

## Позиционирование:
- Лого: `flex-start`
- Навигация: `center` (скрыта на мобильных `hidden md:flex`)
- Кнопки справа: `flex-end`

## Особенности:
- ✅ "Тех.поддержка" и "Подписка" видны **ВСЕГДА** (даже без регистрации)
- ✅ "Загрузить" появляется только после входа
- ✅ "Админ меню" видна только для главного админа
- ✅ При наведении на кнопки навигации - подчеркивание снизу
- 📱 На мобильных вся навигация скрыта
- 🎵 Эмодзи - временная заглушка (заменить на `<img src="/images/logov.png" />`)
