## 🚀 Как работает

### Прослушивания:
```javascript
// В App.tsx
if (progress >= 0.99 && !playCountIncremented) {
  playCountIncremented = true;
  fetch(`/api/sounds/${playingId}/play`, { method: 'POST' });
}
```

### Проверка на бота:
```javascript
// Перетащить ползунок на ≥95% от ширины контейнера
if (newPosition >= container.offsetWidth - 60) {
  setIsVerified(true);
  onVerify('slider_verified_' + timestamp);
}
```

### Фильтр "Моя музыка":
```javascript
userSounds={allSounds.filter(s => s.authorId === store.currentUser?.id)}
```
