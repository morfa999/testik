# Модальные окна приложения

## 1. 🔐 AuthModal - Авторизация/Регистрация
**Открывается:** 
- Кнопка "Войти" или "Начать" в Header
- При попытке незарегистрированного пользователя скачать/добавить звук

**Режимы:**
- **Login** - вход (email + пароль)
- **Register** - регистрация (имя + email + пароль + проверка на бота)

**Особенности:**
- BotCheck компонент (анти-бот проверка)
- Переключение между режимами
- Валидация полей

---

## 2. ➕ AddModal - Добавление звука/пака
**Открывается:** Кнопка "Загрузить" в Header (только для залогиненных)

**Вкладки:**
- **Звук** - загрузка одного аудио файла
  - Поддержка WAV, MP3, FLAC (макс. 50MB)
  - Встроенный AudioEditor
  - Категория, теги, Free/Premium
  - Для обычных пользователей - отправка на модерацию
  - Для админов - публикация сразу

- **Пак** - загрузка нескольких файлов (функция готова, но скрыта в UI)

---

## 3. 👤 ProfileModal - Профиль пользователя
**Открывается:** Клик на аватар в Header

**Функции:**
- Редактирование имени
- Просмотр подписки (none/hd/ultra)
- Статистика скачиваний
- Дата регистрации
- Кнопка "Выйти"

---

## 4. 👑 PremiumModal - Выбор подписки
**Открывается:** 
- Кнопка "Подписка" в Header (доступна всем)
- При попытке скачать Premium звук без подписки

**Тарифы:**
- **Sound HD** - 299₽/мес
  - Premium звуки
  - WAV + MP3
  - До 50 скачиваний/мес
  - Без рекламы

- **Sound Ultra** - 599₽/мес
  - Безлимитные скачивания
  - WAV + MP3 + FLAC
  - Эксклюзивные звуки
  - Коммерческое использование

---

## 5. 📥 DownloadModal - Скачивание звука
**Открывается:** Кнопка "Скачать" на карточке звука

**Форматы:**
- **MP3** - доступен всем
- **WAV** - Sound HD и Ultra
- **FLAC** - только Sound Ultra

**Логика:**
- Если не залогинен → открывает AuthModal
- Если нет нужной подписки → открывает PremiumModal

---

## 6. 🆘 SupportModal - Техподдержка
**Открывается:** Кнопка "Тех.поддержка" в Header (доступна всем)

**Функции:**
- Отправка репорта администраторам
- Доступна без регистрации (но требует входа для отправки)
- Текстовое поле для описания проблемы

---

## 7. 🛡️ AdminPanel - Админ-панель
**Открывается:** Кнопка "Админ меню" (только для `energoferon41@gmail.com`)

**Вкладки:**
1. **Треки** - все опубликованные звуки
   - Прослушивание
   - Удаление
   - Поиск

2. **Запросы** - звуки на модерации
   - Прослушивание
   - Принять/Отказать

3. **Аккаунты** - управление пользователями
   - Назначить/убрать админа
   - Удалить пользователя
   - Главный админ (ДИРЕКТОР) нельзя удалить

4. **Репорты** - сообщения техподдержки
   - Просмотр
   - Пометить как прочитанное

5. **Оповещение** - рассылка уведомлений
   - Заголовок + текст
   - Отправка всем пользователям

---

## 8. 🔔 NotificationsDropdown - Уведомления
**Открывается:** Иконка колокольчика в Header (только для залогиненных)

**Функции:**
- Список уведомлений от админов
- Счетчик непрочитанных (красная точка)
- Пометка как прочитанное

---

## 9. 🎵 AudioEditor - Редактор звука
**Открывается:** Кнопка "Редактировать" в AddModal (после загрузки файла)

**Функции:**
- Визуализация waveform
- Выделение области мышью
- Удаление выделенного фрагмента
- Изменение громкости (0-200%)
- Изменение тональности (-12 до +12 полутонов)
- Реверс аудио
- Предпросмотр с воспроизведением
- Экспорт в WAV

---

## 10. 📊 UploadProgressModal - Прогресс загрузки
**Статус:** Готов к использованию (пока не подключен)

**Функции:**
- Прогресс бар 0-100%
- Статусы: uploading / saving / done / error
- Название файла
- Сообщение об ошибке

---

## 11. 🍪 CookieBanner - Баннер cookies
**Открывается:** Автоматически при первом визите

**Функции:**
- Принять все
- Настроить (обязательные/аналитика/функциональные)
- Отклонить

---

## Приоритет Z-index:
- `z-[200]` - Уведомления (notify toast)
- `z-[110]` - AudioEditor
- `z-[100]` - Остальные модалы
- `z-[50]` - Header
